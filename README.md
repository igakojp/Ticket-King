## FAQ

### It says the bot is thinking forever.
Please check if the Ticket King bot has administrator permission in the role, and put the Ticket King role above most roles.
### How do I change the profile picture, name, and the banner?
You can not change them, since that is not a feature yet. But you can change the name using Discord's renaming feature.
### How do I make a ticket panel?
Please look at [this tutorial](https://github.com/igakojp/Ticket-King/blob/main/Step%20by%20step%20on%20how%20to%20create%20a%20ticket%20panel.md).
